Analysis of $PM_{2.5}$ Air Pollution in BhopalThis repository contains a computational physics project focused on analyzing real-time air quality data for Bhopal, India. By utilizing Python-based data processing and visualization, the project extracts physical insights into pollution dynamics and evaluates local air quality against international health standards.## Project OverviewThe primary goal of this study is to analyze $PM_{2.5}$ (particulate matter with diameter $\le 2.5\ \mu m$) levels using historical data from Central Pollution Control Board (CPCB) monitoring stations.### Key ObjectivesData Acquisition: Sourcing authentic daily $PM_{2.5}$ measurements from Bhopal.Computational Processing: Cleaning, processing, and visualizing data using the Python ecosystem.Physical Interpretation: Applying concepts such as diffusion and thermodynamics to understand pollution behavior.Trend Identification: Identifying seasonal peaks, periodic cycles, and extreme pollution events.## FeaturesThe provided Python script (CP grp project.py) performs several layers of analysis:Statistical Characterization: Calculation of mean, maximum, and standard deviation to assess pollution severity.Time-Series Analysis: Visualization of raw $PM_{2.5}$ fluctuations over time.Moving Average Smoothing: Implementation of a 7-day rolling average to reveal underlying seasonal trends.Frequency Domain Analysis: Utilizing Fast Fourier Transforms (FFT) to identify periodic cycles linked to industrial and traffic patterns.Distribution Analysis: Histogram generation to understand the frequency of moderate vs. extreme pollution events.## Key FindingsMetricValueInsightMaximum$1000\ \mu g/m^3$Extreme pollution spikes detected.Mean$80.27\ \mu g/m^3$$5
